get /api/snh48/
searches for all team members (searchInventory)
By passing in the appropriate options, you can search for all current memeber of SNH48 in the system

Query parameters

member/id (optional)
Query Parameter — use an id an to retrieve data
member?name (optional)
Query Parameter — use name of the member to retrieve data
member?team (optional)
Query Parameter — use the team name to retrieve data

Return type

array[SNH48Model]
